# Brand Logos

Place the brand logo images in this directory with the following filenames:

1. **mrassistant-logo.png** - Logo for mrassistant.ai
2. **buzzzbuzzz-logo.png** - Logo for buzzzbuzzz.com
3. **growthlab-logo.png** - Logo for growthlab.sg
4. **onestopsg-logo.png** - Logo for onestopsg.com

## Recommended Specifications:
- Format: PNG (with transparency) or SVG
- Size: Minimum 240x240px (will be scaled down)
- Background: Transparent preferred
- Aspect Ratio: Square or landscape (will be contained)

The logos will be displayed at approximately 120x120px on desktop and scaled responsively on mobile.
